OBIETTIVO DEL PROGETTO
Gestire i libri con l'utilizzo di un ArrayList.
I dati vengono caricati da un file di testo.
La classe generica utilizzata per la lettura prevede l'indicazione della classe da 
utilizzare come parametro nella costruzione dell'arrayList.
